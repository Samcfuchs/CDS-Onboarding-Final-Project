Visualizing Linear Regression

Use the arrow keys to navigate through the epochs of gradient descent, eventually converging on the final, optimized model. On the left is plotted the original dataset, with the current regression model shown in red. On the right, each model is shown plotted by its a and b coefficients (of the form a + bx), and colored by the mean squared error of the model, which we use as the loss function of the gradient descent.
